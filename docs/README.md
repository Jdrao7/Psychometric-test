# Database Documentation

This folder contains SQL schemas and documentation for the Supabase database.

## Setup Instructions

1. Go to your Supabase project dashboard
2. Navigate to **SQL Editor**
3. Run the SQL files in order:
   - `schema.sql` - Creates all tables

## Tables Overview

| Table | Purpose |
|-------|---------|
| `assessment_results` | Candidate assessment scores and insights |
| `custom_roles` | Recruiter-created role criteria |
| `candidate_role_matches` | Color-coded candidate-role matching |
